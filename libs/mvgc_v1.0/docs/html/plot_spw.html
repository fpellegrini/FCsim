plot_spw

Plot spectral pairwise quantities on a grid

code

Syntax

   plot_spw(P,fs)

Arguments

input

   P          matrix of spectral pairwise quantities
   fs         sample rate in Hz (default: normalised freq as per routine 'sfreqs')
   frange     frequency range to plot: empty for all (default) else an ascending 2-vector

Description

Plot pairwise spectral quantities in P, a 3-dim numerical matrix with first index representing target ("to"), second index source ("from") quantities and third index frequencies - typically spectral causalities (see e.g. autocov_to_spwcgc).

See also

autocov_to_spwcgc | mvgc_demo | sfreqs
